# Abalone Age Prediction & Statistical Analysis

## Overview
This project analyzes the Abalone dataset to predict age using statistical and machine learning techniques.

## Objectives
- Clean and validate dataset
- Explore relationships between variables
- Detect multicollinearity
- Build regression and classification models
- Apply PCA for dimensionality reduction
- Perform clustering analysis

## Methods Used
- Data Cleaning (handling invalid values)
- Exploratory Data Analysis (EDA)
- Correlation Matrix & VIF
- Linear Regression (OLS)
- Principal Component Analysis (PCA)
- Logistic Regression
- K-means & Hierarchical Clustering

## Key Results
- Regression model R² ≈ 0.52
- Test MSE ≈ 4.64
- Strong multicollinearity detected
- PCA reduced correlation while maintaining performance
- Logistic regression showed bias toward male class

## Tools & Libraries
- Python
- pandas
- numpy
- seaborn
- matplotlib
- scikit-learn
- statsmodels
